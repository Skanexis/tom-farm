
  # Tom Farm

  Tom Farm catalog app with React frontend and Express API.

  ## Local development

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  In another terminal, run `npm run server` to start the API.

  ## Production

  Run `npm run build` and then `npm run start`.

  For VPS deployment with Git, Docker and Nginx, see `DEPLOY.md`.
